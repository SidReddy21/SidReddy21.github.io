# Sid Reddy Personal Website

This repo contains the personal website for Sid Reddy.
The public website link is: [sidreddy21.github.io](https://sidreddy21.github.io/)

## Installation & Usage
To run the website locally, first clone the repo using the following command

```bash
git clone https://github.com/SidReddy21/SidReddy21.github.io.git
```

Enter the project

```bash
cd sidreddy21.github.io
```

Install necesarry dependencies

```bash
npm install
```

Run locally

```
npm run dev
```
